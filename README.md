## Installation

```sh
git clone https://github.com/kingisaac95/react-webpack-starter.git
```

Change directory

```sh
cd react-webpack-starter
```

Install Package dependencies

```sh
npm install
```

Run the application

```sh
npm start
```

Run test

```sh
npm run test
```
